Introduction

This section describes how this website is structured and gives guidelines to maintain it.

This website has been created with Github pagesand is hosted on github.com. It uses Jekyll's platform to transform text files into HTML websites. This allows the users to focus on the content, rather than on the site's source code. Pages are written in the very simple Markdown language (and can also include snippets of HTML).
