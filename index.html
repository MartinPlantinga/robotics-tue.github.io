--- layout: "home" author_profile: false --- Welcome to the Robotics website of the Control Systems Technology Group of the Eindhoven University of Technology.

This website centralizes software development of the Control Systems Technology group that is related to robotics.

In this website you can find:
  • Information about the group
  • Access to repositories
  • Tutorials
  • Information about the Robotics Laboratory
The focus of this website is to keep code and documentation together in one place. In the repositories section users and developpers can find the repositories that are being or have been developped within the group. Each repository can contain a wiki for its documentation.

Students and Employees are encouraged to enrich this website by adding and/or updating content. Content is written in simple Markdown files. More information about the maintenance of this site can be found in the maintenance section.

Information about the structure and maintenance of this site can be found in the README file of this website's repository.
