--- layout: "home" author_profile: false --- Welcome to the Robotics website of the Control Systems Technology Group of the Eindhoven University of Technology.

This website centralizes software development within the Control Systems Technology group related to robotics.

In this website you can find:
  • Information about the group
  • Access to repositories
  • Tutorials
  • Information about the Robotics Laboratory

Each repository contains a Wiki. Make use of this feature to document the repositories.

Students and Employees are encouraged to enrich this website by adding and/or updating content. Content is written in simple Markdown files. A useful Markdown tutorial can be found at www.markdowntutorial.com.

Information about the structure and maintenance of this site can be found in the README file of this website's repository.
